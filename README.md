# Redux Typed Ducks - Saga example

This project tries to show a proper usage of [redux-saga](https://redux-saga.js.org/) in conjunction with [Redux Saga](https://github.com/mhoyer/redux-typed-ducks).

## Run it

    npm i # or yarn
    npm run dev
